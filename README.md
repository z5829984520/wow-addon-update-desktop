##### 使用说明

1. 第一次使用无法获取到你wow已经安装的插件，技术能力有限
2. 在插件列表找到你安装过的可以再次在这个工具里安装一次就会有历史安装插件记录方便更新和移除
3. 确保你的网络能访问到[curse](https://www.curseforge.com/wow/addons)
4. 此工具个人使用，请不要使用商业上
